This plugin allows you to duplicate your menus from backend, simply select the menu which you want to duplicate from  plugins settings page and input new menu name, just click on "Duplicate Menu" button, and its done :)

== Installation ==
1. Upload the plugin files to the `wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
